Android中userId、uid、appId区别(基于Android 7.0源码)

android中的uid和linux系统中的uid是两码事，linux系统中的uid是针对多用户操作系统中用来区分用户的；android中借鉴了linux的思想，对于uid进行了改变，在android中uid是分配给各个app使用的，每个app的uid是唯一的，用来做权限管理的。
参考源码路径：
/frameworks/base/core/java/android/os/UserHandle.java

  • userId：是指用户Id；多用户表示，单用户都是0

  • appId： 是指跟用户空间无关的应用程序id；取值范围 0<= appId <100000；

  • uid: 是指跟用户空间紧密相关的应用程序id;

三者之间有下面的转换关系

UserHandle源码解读

public static final int PER_USER_RANGE = 100000;//每个用户下面最大可以有100000 个appId

Process源码解读

  • system appid: [1000, 9999]

  • application appid:[10000, 19999]

  • isolated appid: [99000, 99999]
